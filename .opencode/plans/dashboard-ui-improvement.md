# Dashboard UI 改进计划 - 高端军事情报风格

## 当前问题

1. **视觉效果太简单**：没有层次感，缺少深度
2. **缺少动态效果**：没有微交互动画，显得死板
3. **数据可视化不够突出**：图表样式普通，没有科技感
4. **没有"军事/情报系统"氛围**：缺少 HUD 元素、扫描线、角标装饰

## 改进方向

### 1. 背景层（Background Layers）
- **多层背景**：
  - 底层：深色渐变（#0a0a0b → #1a1a1d）
  - 中层：网格线（rgba(0, 255, 136, 0.03)）
  - 顶层：扫描线动画（从上到下缓慢移动）
- **Logo 水印**：VeriSure Logo 作为背景水印（opacity: 0.02）
- **粒子效果**：可选，添加微弱的粒子动画

### 2. 卡片设计（Card Design）
- **HUD 风格边框**：
  - 四角装饰（L 型角标）
  - 边框发光效果（box-shadow: 0 0 20px rgba(0, 255, 136, 0.2)）
  - 扫描线动画（从上到下缓慢移动）
- **玻璃态效果**：backdrop-filter: blur(10px)
- **悬停效果**：
  - 边框颜色变化（rgba(0, 255, 136, 0.3) → rgba(0, 255, 136, 0.6)）
  - 轻微上移（transform: translateY(-4px)）
  - 发光增强

### 3. 数据可视化（Data Visualization）
- **统计面板（StatPanel）**：
  - 大号数字（font-size: 48px, font-weight: 700）
  - 数字滚动动画（从 0 到目标值）
  - 标签使用等宽字体（JetBrains Mono）
  - 四角装饰 + 扫描线
- **雷达图**：
  - 使用霓虹绿色（#00ff88）
  - 添加发光效果
  - 网格线使用半透明绿色
- **饼图/环形图**：
  - 使用渐变色
  - 添加发光边框
  - 中心显示总数
- **折线图**：
  - 使用渐变填充
  - 添加数据点发光效果
  - 网格线使用半透明绿色

### 4. 动画效果（Animations）
- **数字滚动**：统计数字从 0 滚动到目标值
- **扫描线**：卡片顶部有缓慢移动的扫描线
- **脉冲效果**：状态指示灯有脉冲动画
- **悬停动画**：卡片悬停时有平滑过渡
- **加载动画**：数据加载时显示骨架屏

### 5. 色彩方案（Color Scheme）
- **主色**：霓虹绿（#00ff88）
- **辅助色**：电光蓝（#00b4d8）、警告橙（#ff6b35）、危险红（#ff4d4f）
- **背景**：深色（#0a0a0b, #1a1a1d）
- **文字**：白色（rgba(255, 255, 255, 0.9)）和半透明白色（rgba(255, 255, 255, 0.6)）
- **字体**：
  - 标题：系统默认字体
  - 数据：JetBrains Mono（等宽字体）
  - 标签：JetBrains Mono

### 6. 布局改进（Layout Improvements）
- **顶部导航栏**：
  - 左侧：Logo + 品牌名称 + 分类标签（CLASSIFIED // TOP SECRET）
  - 中间：组织切换器（下拉菜单）
  - 右侧：用户信息 + 设置按钮 + 退出按钮
- **统计面板区域**：
  - 4 个面板横向排列
  - 每个面板有独立的角标装饰
- **图表区域**：
  - 3 个图表横向排列（雷达图、饼图、折线图）
  - 每个图表有标题和副标题
- **项目卡片区域**：
  - 网格布局（3 列）
  - 每个卡片有 Dossier 风格
  - 悬停时显示更多详情

### 7. 交互改进（Interaction Improvements）
- **组织切换**：点击组织名称，下拉菜单显示所有组织
- **项目卡片**：点击卡片跳转到项目详情页
- **筛选/排序**：使用下拉菜单和搜索框
- **刷新按钮**：点击刷新数据，显示加载动画
